# 引文

说到写 Java，想必做 Java 开发的童鞋，蛮多都是会用 IntelliJ IDEA， Eclipse 或是 Netbeans。许多童鞋也许会问，VS Code 也能写 Java? 是的，没错，VS Code 可以写 Java，而且有着很好的开发体验。特别对于多语言开发者来说，如果能在同一个 IDE 或是编辑器编写多种语言，岂不是很方便？笔者在微软近六年，除了C#，主要写 TypeScript，还写过 PHP，Python，Java 等语言。自从有了 VS Code，再也不用在不同的 IDE/编辑器之间切换了。那么，我们就来看看 VS Code 中有哪些好用的 Java 插件。

# [Language Support for Java(TM) by Red Hat](https://marketplace.visualstudio.com/items?itemName=redhat.java)

![20200831230142](https://liulv.work/images/img/20200831230142.png)

Red Hat 的 Java 插件是每个 VS Code Java 用户的必装插件。对 Maven/Gradle 项目的基本支持、代码格式化，代码重构、代码片段、语法高亮、代码自动补全等等核心功能，都在这个插件中。
[
# [Debugger for Java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-debug)

![20200831230401](https://liulv.work/images/img/20200831230401.png)

调试对于每一个语言来说当然是必不可少的功能。Debugger for Java 除了提供 Launch/Attach、断点/条件断点、Step In/Out/Over、Callstacks 这些基本功能，还支持 Logpoints、Hot Code Replace 等高级功能，使得调试 Java 项目如丝般顺滑。

# [Java Test Runner](https://link.zhihu.com/?target=https%3A//marketplace.visualstudio.com/items%3FitemName%3Dvscjava.vscode-java-test)

在项目生命周期中，测试是一个重要的环节。Java Test Runner 对主流的测试框架 JUnit 和 TestNG 都有着很好的支持。在 Test Explorer 中，可以方便地查看所有测试用例、运行/调试测试代码、查看测试报告。

# [Maven for Java](https://link.zhihu.com/?target=https%3A//marketplace.visualstudio.com/items%3FitemName%3Dvscjava.vscode-maven)

Maven 是 Java 生态圈中使用最多的构建工具，在 VS Code 中也有很好的支持。基于 Maven Archetype 生成 Maven 项目、运行各种 Maven 命令，在 VS Code 中也能轻松地开发 Maven 项目。

# [Java Dependency Viewer](https://link.zhihu.com/?target=https%3A//marketplace.visualstudio.com/items%3FitemName%3Dvscjava.vscode-java-dependency)

顾名思义，Java Dependency Viewer 可以方便地列出当前 Java 项目所有地依赖项，并且可以开始地创建一个 Java 项目。
